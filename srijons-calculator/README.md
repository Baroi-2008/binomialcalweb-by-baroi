# Srijon's Calculator

A modern, sleek Quadratic Equation Solver web app.

## Features
- **Modern Dark Mode**: Glassmorphism design with neon accents.
- **Responsive**: Works on Mobile, Tablet, and Desktop.
- **Instant Results**: Calculates roots immediately.

## How to Upload to GitHub & Host (Free)

1.  **Create a GitHub Account** (if you don't have one) at [github.com](https://github.com).
2.  **Create a New Repository**:
    *   Click the "+" icon in the top right -> "New repository".
    *   Name it `srijons-calculator`.
    *   Check "Add a README file".
    *   Click "Create repository".
3.  **Upload Files**:
    *   Click "Add file" -> "Upload files".
    *   Drag and drop `index.html`, `style.css`, and `script.js` into the box.
    *   Click "Commit changes".
4.  **Activate GitHub Pages**:
    *   Go to "Settings" tab in your repository.
    *   Scroll down to "Pages" (on the left sidebar).
    *   Under "Source", select `main` (or `master`) branch.
    *   Click "Save".
5.  **Done!**
    *   Wait a minute, and you will see a link like `https://your-username.github.io/srijons-calculator/`.
    *   Share this link! You can open it on your phone and "Add to Home Screen" to use it like an app.
